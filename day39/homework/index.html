Understanding Color Systems: RGB, HEX, HSL

RGB Color System

RGB stands for Red, Green, Blue. Colors are made by mixing these three colors in different amounts.

HEX Color System

HEX stands for hexadecimal. It's a way to write colors using six characters, representing red, green, and blue.

HSL Color System

HSL stands for Hue, Saturation, and Lightness. It represents colors based on their shade and brightness.
